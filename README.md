# Interior Decor e-Shop

## Project Description
The Interior Decor e-Shop is a simple HTML5 website designed to showcase interior decoration products and services. It provides visitors with information about home decor collections, shopping guides, services, and contact details through a clean and user-friendly interface.

## Features
- Home page with a welcome section
- Navigation menu
- Featured room inspiration
- Interior decor collections
- Product price table
- Ordered, unordered, and description lists
- Services section
- Contact information
- Internal page navigation
- Responsive layout using basic HTML5 and CSS
